UP9 - Automatically Generated Postman Tests


Requirements:

Postman needs to be installed on your machine: https://www.postman.com/downloads/

Running the tests:

Open Postman, click "Import" and select `UP9.postman_collection.json` from your file system.

Data Driven Tests:

To parameterize your tests follow this Postman guide https://learning.postman.com/docs/running-collections/working-with-data-files/
and select the `dataset_` prefixed JSON files under the `data/` directory as your data files.
